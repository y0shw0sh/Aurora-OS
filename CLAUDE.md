# Project Rules

## Architecture
- Next.js App Router
- TypeScript
- Tailwind
- Motion
- Supabase

## UI
- Apple VisionOS inspired
- translucent
- smooth
- premium
- minimal

## Code
- modular
- reusable
- no giant files
- maximum ~300 lines per component

## Animations
- spring
- 60fps
- layout transitions

## Never
- rebuild working features
- delete existing code
- create placeholder implementations

## After every change
- explain files changed
- explain commands to run
- explain how to test
